# Vercel Deployment Instructions

To deploy this application to Vercel, use the following settings:

1.  **Project Name:** `normtrace-ihr-mexico`
2.  **Framework Preset:** `Vite`
3.  **Root Directory:** `05_webapp`
4.  **Build Command:** `npm run build`
5.  **Output Directory:** `dist`
6.  **Install Command:** `npm install`

### Steps:
1.  Import the repo into Vercel.
2.  Set `Root Directory` to `05_webapp`.
3.  Deploy.
