# NormTrace-IHR Methodology (Web Version)

## What problem NormTrace-IHR solves
- SPAR and JEE are useful monitoring instruments but do not reconstruct domestic legal anchoring obligation by obligation.
- SPAR is standardized and recurrent but self-reported.
- JEE, AAR and SimEx add external or functional assessment layers but do not map each international obligation to domestic legal authority, actors, procedures, coordination and safeguards.
- NormTrace-IHR adds a country-specific legal-institutional conversion layer.

## Core contribution
> **NormTrace-IHR is not a compliance score. It is a legal-institutional traceability infrastructure that connects international health obligations to domestic legal texts, actors, competences, procedures and review needs.**

## Conceptual workflow
1. **International obligation classification**
2. **Domestic legal corpus construction**
3. **Country-specific legal system profile / legal brain**
4. **Legislative drafting pattern extraction**
5. **Domestic provision extraction**
6. **Actor and competence mapping**
7. **Legal anchoring assessment**
8. **Gap typology**
9. **Capacity-building entry points**
10. **IHR 2024 and Pandemic Agreement update layers**

## Why this is not keyword matching
The system first reconstructs Mexico’s constitutional, federal, administrative, health-governance and regulatory architecture before matching obligations to domestic provisions.

## Mexico pilot scope
- 45 IHR 2005 obligation units
- 110 domestic legal provisions
- 18 domestic instruments
- 80 mapping rows
- 18 profiled actors
- federal-level corpus only
- preliminary expert-review version

## Anchoring scale
| Level | Definition |
| :--- | :--- |
| **0** | No identifiable legal-institutional anchor |
| **1** | Actor mention only |
| **2** | Indirect anchoring through broad mandate language |
| **3** | Administrative or sub-statutory anchoring |
| **4** | General statutory anchoring |
| **5** | Specific legal-institutional anchoring |

*Note: The scale measures identifiable legal-institutional anchoring in the available corpus. It does not measure compliance, implementation performance or operational effectiveness.*

## Gap typology
| Gap Type | Description |
| :--- | :--- |
| legal silence | No identifiable provision |
| competence ambiguity | Unclear institutional responsibility |
| administrative-only anchoring | Lacks statutory basis |
| procedural gap | Missing implementation steps |
| coordination gap | Missing inter-institutional links |
| federal implementation gap | Jurisdictional misalignment |
| rights-safeguard gap | Missing accountability or protections |
| oversight gap | Missing monitoring or review |
| budget/capacity gap | Missing financial or human resources |
| update-review needed | Alignment needed with IHR 2024 or Pandemic Agreement |

## Role of Python, AI and human review
- **Python** is used for cleaning, validation, cross-reference checks, network analysis and reproducibility.
- **AI** is used for assisted structuring, metadata coding, pattern detection, provision extraction and preliminary classification.
- **AI is not used as legal authority.**
- **Human expert review** remains required for legal validity, interpretation and policy use.
